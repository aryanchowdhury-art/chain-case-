What This Covers


Exploratory Analysis – Order status breakdown, monthly sales trends, top products by revenue, customer segment performance.
Supply Chain Metrics – Late delivery rate by shipping mode, lead time by category, order fill rate by region, discount impact on profitability.
Advanced SQL – Rolling 3-month revenue using window functions, product ranking within categories, underperforming region detection via CTEs/subqueries, customer lifetime value ranking.
Open-Ended Analysis – Supplier performance dashboards, year-over-year category growth, shipping delay anomaly detection, profitability by geography.


SQL Concepts Applied

Multi-table JOINs, aggregate functions (SUM, AVG, COUNT DISTINCT), GROUP BY/HAVING, CASE WHEN logic, Common Table Expressions (CTEs), subqueries, and window functions (RANK(), DENSE_RANK(), rolling averages with ROWS BETWEEN).

Why This Project

Used a real-world supply chain dataset (~180K rows) to practice moving from simple exploratory queries to complex, multi-step analytical SQL — identifying late-delivery risk, underperforming regions, and revenue trends the way a working data analyst would.
